Phishing Awareness Simulator

Una simulación que muestra ejemplos de correos electrónicos o páginas falsas diseñadas para educar a los usuarios sobre cómo detectar intentos de phishing.

Esta simulación presenta ejemplos de correos electrónicos falsos o páginas web diseñadas para parecerse a los originales. Los usuarios interactúan con ellos y deben identificar señales de phishing como errores ortográficos, enlaces sospechosos o solicitudes de datos personales. El sistema ofrece explicaciones detalladas sobre por qué ciertos elementos son peligrosos, ayudando a los usuarios a reconocer intentos reales en el futuro.
